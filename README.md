# Android ChatUIPractice 聊天界面练习
一个简单的Android UI聊天界面练习，使用Kotlin + Android Studio 4.1.2开发，使用了RecyclerView来显示消息，使用RecyclerView + GridLayoutManager显示Emoji表情选择，实现了消息时间逻辑。
